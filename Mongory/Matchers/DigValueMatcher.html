Overview

DigValueMatcher is responsible for extracting a value from a record using a key (or index) and then delegating the match to DefaultMatcher logic.

It handles nested access in structures like Hashes or Arrays, and guards against types that should not be dig into (e.g., String, Symbol, Proc).

This matcher is typically used when the query refers to a specific field, like { age: { :$gte => 18 } } where :age is passed as the dig key.

Examples:

matcher = DigValueMatcher.build(:age, { :$gte => 18 })
matcher.match?({ age: 20 }) #=> true
