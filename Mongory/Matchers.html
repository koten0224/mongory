Overview

Defines built-in Mongory matchers for query condition evaluation.

This file loads and registers all available matcher classes and maps Mongo-style operators (e.g., $gt, $in) to matcher implementations. Matcher lookup and operator dispatch mapping.

This module contains the $operator => MatcherClass mapping used by HashConditionMatcher to find the correct matcher for a condition.

Matchers are loaded from the matchers/ directory.

def self.lookup(key)
  return unless OPERATOR_TO_CLASS_MAPPING.include?(key)

  const_get(OPERATOR_TO_CLASS_MAPPING[key])
end
